fs-tabnav-item
directive in module fs

Usage

as element:
<fs-tabnav-item
       fs-url="{string}"
       fs-name="{string}"
       fs-disabled="{string}"
       fs-click="{function}">
</fs-tabnav-item>

Parameters

ParamTypeDetails
fs-urlstring

The url that the browser will be redirected to when the tab is clicked

fs-namestring

The name of the tab used for selecting the active tab through fs-tabnav.fs-selected

fs-disabledstring

When set to true the tab will be in a disabled state

fs-clickfunction

A function that is fired when the tab is clicked
